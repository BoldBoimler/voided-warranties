# Voided Warranties

Things I've taken apart.

## What's this?

I take things apart.

Sometimes I can't find anyone else who’s done it—especially when I just want to know something simple like:
- are there TX/RX pins?
- is anything reusable?

So this is basically a collection of photos and notes from whatever I end up opening.

## Why?

Sometimes you just need to look at something's guts to understand it better.  Also good to know if you're going to break something along the way.
